# 🍦 Creamery App

## 📌 Project Overview

Creamery App is a simple and interactive web application built using React that showcases different ice cream products and desserts. The application provides users with a clean user interface to explore menu items and learn more about the creamery.

This project was developed as a learning project to practice **React components, routing, and frontend design**.

---

## 🚀 Features

* Responsive user interface
* Interactive navigation
* Ice cream menu display
* Product cards with details
* Simple and clean design

---

## 🛠️ Technologies Used

* React JS
* JavaScript
* HTML5
* CSS3
* Vite

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/YalapalliKavya/Creamery-app.git
```

2. Navigate to the project folder

```
cd Creamery-app
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```

The application will start and open in your browser.
